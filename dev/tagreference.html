見出し <h1>～<h6>

<h1>～<h6>は見出しを表現します。数字が小さいほど大きな見出しとなります。

<h1>1番目に大きな見出し</h1>
<h2>2番目に大きな見出し</h2>
<h3>3番目に大きな見出し</h3>
<h4>4番目に大きな見出し</h4>
<h5>5番目に大きな見出し</h5>
<h6>6番目に大きな見出し</h6>

コード例

<h1>JavaScript</h1>
<h2>特徴</h2>
<h2>歴史</h2>
<h3>誕生</h3>
<h3>現在</h3>
<h3>JavaScript 2.0</h3>
<h2>セキュリティ上の制限</h2>
<h2>文法</h2>

※コード例はWikipediaのJavaScriptの項目の見出し。

段落・改行 <p>, <br>

<p>はひとつの段落（パラグラフ）を表現します。

<p>最初の段落です。</p>
<p>次の段落です。</p>

コード例

<p>吾輩は猫である。名前はまだ無い。</p>
<p>どこで生れたかとんと見当がつかぬ。</p>

<br>は改行を表現します。文の長さを揃えたりレイアウトを整えるなどの目的に使用することは控えましょう。

<p>この文字の後で<br>改行されます。</p>

コード例

<p>天は人の上に人を造らず<br>人の下に人を造らずと言えり。</p>

ハイパーリンク <a>

<a>はハイパーリンクを表現します。href属性にはリンク先のURLか、リンクしたいファイルの相対パスを記述します。
<a>で囲まれた文字や画像（<img>で記述する）をクリックするとリンク先にジャンプさせることができます。

<a href="リンク先">リンクしたい文字や画像</a>

コード例

<a href="http://www.google.co.jp/">Google検索</a>
<a href="index.html">トップページに戻る</a>

画像 <img>

<img>は画像を埋め込みます。src属性に埋め込みたい画像のファイルの相対パスやURLを記述します。

<img src="画像ファイルの相対パスやURL">

コード例

<img src="img/logo/logo-large.png">
<img src="http://twitter.com/images/resources/twitter-bird-light-bgs.png">

順序なし・順序あり リスト <ul>, <ol>

<ul>は順序なしのリスト、<ol>は順序ありのリストを表現します。
リスト項目は、どちらの場合も<li>タグで指定します。

<ul>
  <li>順序なしのリスト項目</li>
  <li>順序なしのリスト項目</li>
  <li>順序なしのリスト項目</li>
</ul>

<ol>
  <li>順序ありのリスト項目</li>
  <li>順序ありのリスト項目</li>
  <li>順序ありのリスト項目</li>
</ol>

コード例

<ul>
  <li>お袋</li>
  <li>給料袋</li>
  <li>堪忍袋</li>
</ul>

<ol>
  <li>零号機</li>
  <li>初号機</li>
  <li>弐号機</li>
</ol>

定義・説明 リスト <dl>

<dl>は、定義・説明リストを表現します。<dl>と</dl>の中に、<dt>で定義・説明される言葉を、<dd>でそれに対応する説明を配置してリストを作成します。
用語の定義リストを作成したり、何かの手順を箇条書きで説明する場合などに使用します。

<dl>
  <dt>言葉1</dt>
  <dd>言葉1に関する説明</dd>
  <dt>言葉2</dt>
  <dd>言葉2に関する説明</dd>
  <dt>言葉3</dt>
  <dd>言葉3に関する説明</dd>
</dl>

コード例

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>JK</dt>
  <dd>常識的に考えて</dd>
  <dt>チョベリバ</dt>
  <dd>超ベリーバッド</dd>
  </dl>

フォームの部品 <input>

<input>はユーザからの入力を受け付けるフォームの部品を表現します。
type属性を指定することで、フォームの形式を変化させることができます。また、value属性を指定することで、フォームの入力値（ボタンに設定した場合はボタンのラベル表示）を設定することができます。

<input type="フォームの形式" value="フォームの入力値（省略可能）">

以下に代表的なtype属性の設定値の一覧を示します。

type属性の設定値 表示されるフォームの形式
text 一行テキストボックス（初期値）
button 汎用ボタン
checkbox チェックボックス
radio ラジオボタン
password パスワード入力欄

コード例

<!-- テキスト入力ができるidがaというフォーム部品 -->
<input id="a" type="text">

<!-- idがaのOKと書かれたボタン -->
<input id="b" type="button" value="OK">

複数行のテキスト入力欄 <textarea>

<textarea>は複数行のテキスト入力を受け取るフォームの部品を表現します。
cols属性で一行当たりの最大文字数の目安、rows属性で表示行数を指定します。

<textarea cols="一行あたりの最大文字数の目安" rows="行数">

以下に代表的な属性の一覧を示します。

属性 解説
maxlength 入力可能な最大文字数を指定する。
placeholder ユーザー入力に関するヒントを指定する。
wrap 入力テキストの改行ルールを指定する。softかhardの二種類で、hardを指定する場合には、同時にcols属性を指定する。
readonly ユーザーによるテキスト編集を不可にして読み取り専用にする場合に属性名を書く。値は指定する必要はない。

コード例

<!-- 感想の入力フォームを表示する -->
<textarea cols="40" rows="4" maxlength="20" placeholder="感想をどうぞ">

共通属性 id, style, class, hidden, title

以下に示す属性はすべてのHTML要素に共通で指定することができます。

属性 解説
id ページ内でユニークな識別子(ID)を指定します。空白文字を入れたIDを指定することはできません。CSSやJavaScriptでHTML要素を特定したい場合に利用します。
style HTML要素の見た目を変更するために、CSSを記述するための属性です。
class CSSで定義したクラスを適用する時に利用します。CSSには「.クラスの名前」としてスタイルを定義し、class属性の値としてクラス名を指定します。
hidden この属性があるHTML要素はWebブラウザに表示されません。属性の名前だけを記述すればよく、値は指定する必要はありません。
title そのHTML要素のツールチップとなる補足情報を表します。HTML要素がリンクであれば、リンク先のリソースの表題や説明、画像であれば、その画像の著作権表示や説明になります。

イベント属性 onclick, onchange等

<input>をはじめとしたHTMLの要素には、様々なイベントハンドラ（あるイベントが発生したことをきっかとして、指定した処理を実行するもの）を設定することができます。フォームの部品にイベントハンドラを設定し、指定したJavaScriptの関数を実行させる方法が多く使われます。

Hello!というボタンを表示し、そのボタンがクリックされたら、hello_button_clickedという名前のJavaScriptの関数を実行するコード例は以下のようになります。

<input type="button" value="Hello!" onclick="hello_button_clicked()">

以下にonclickも含めた、主なイベントハンドラの例を示します。

タイプ イベント イベントの内容
ページ等 onload ページ等の読み込み完了した時
onunload 他のページへの移動時、ページのクローズ時
onresize サイズが変更された時
onscroll スクロールした時
マウス onclick クリックした時
ondblclick ダブルクリックした時
onmousedown マウスボタンを押した時
onmouseup マウスボタンを放した時
onmouseover マウスが重なった時
キー onkeydown キーが押し下げられた時
onkeypress キー入力があった時
onkeyup 押し下げられたキーが放された時
フォーム onchange 内容が変更された時
onsubmit サブミットボタンが押された時
onreset リセットボタンが押された時
フォーカス onfocus フォーカスされた時
onblur フォーカスが外れた時

範囲指定 <div>, <span>

<div>や<span>は範囲を指定するために使います。id属性・class属性・style属性などと一緒に使用します。
divは指定した範囲が改行され、spanは改行されません。

<div 属性名="属性の値">範囲を指定したい部分（改行される）</div>
      <span 属性名="属性の値">範囲を指定したい部分</span>

コード例

<div id="answer" class="red">
  この範囲にanswerというidを設定し、CSSのredというクラスを適用します。
</div>

<p>これは<span style="background-color: #0099FF">青色</span>です。</p>

メタデータ <head>, <meta>, <title>, <link>, <style>

HTMLに関する情報をメタデータと呼び、複数のメタデータは<head>で囲んで表現します。

タグ 解説
<head> HTMLのヘッダ情報を表現します。以下のtitle、link、meta、styleなどのタグをこのタグの中に記述します。
<meta> その文書に関する情報（メタデータ）を指定します。charset属性でページの文字コードを指定する場合に使われます。そのほかにも検索エンジン向けのキーワードなども指定できます。
<title> HTMLのタイトルを付けます。
<link> リンクする外部リソースを指定します。
<style> スタイルシートを記述します。

以下にいくつかメタデータを定義したHTMLの例を示します。

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>はじめてのHTMLページ</title>
    <link rel="stylesheet" href="default.css">
  <head>
  <body>
    <p>ページの本文</p>
  </hody>
</html>

コメント <!-- -->

<!--と-->で囲まれた部分はHTMLのソース中のコメントを表現します。一時的にコンテンツの一部を隠したり、タグを無効にする際にも使用します。

<!-- コメント -->

コード例

<!-- HTMLの文字コードはUTF-8にする -->
<meta charset="UTF-8"/>
